# 早知：每日政治经济简报

一个可公开访问的中文简报页面，聚焦中国政策和经济、美国市场与比特币。页面优先展示短标题、阅读框架、原文链接和非实时市场报价，避免复制受版权保护的新闻正文。

## 自动更新

GitHub Actions 每天 `08:07` 北京时间运行一次。GitHub 的定时任务可能排队几分钟，页面会显示实际完成时间。

数据构建脚本会：

- 通过 Google 新闻 RSS 获取中国经济、美国宏观、贸易、美股与 BTC 的公开新闻线索。默认优先 Reuters、Bloomberg、Financial Times、Nikkei Asia、WSJ、AP、BBC 等海外媒体；中国官方发布只作为原始口径核对，不以中国财经门户或转载站作为自动补位；
- 将英文标题翻译为中文，保留原文标题供核对；每条内容同时给出一句话摘要、阅读判断、资产影响和下一观察点。解读基于公开标题和页面市场快照，不替代原文阅读或投资建议；
- 通过 Yahoo Finance 的公开图表接口获取市场最近可得报价；
- 把结果写入 `data/brief.json`，提交至 `main` 分支，供 GitHub Pages 自动发布。

所有新闻条目都保留原始来源链接。页面中的“为什么重要”“可能影响”“仍不确定”是通用阅读框架，不是投资建议，也不替代对原文和官方数据的核对。

## 本地更新

```bash
python3 scripts/build_brief.py
```

随后在浏览器中打开 `index.html`，或在目录中启动任意静态文件服务器。
